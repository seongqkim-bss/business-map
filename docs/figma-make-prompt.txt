아래 JSON 토큰을 기준으로 bss.or.kr 스타일의 페이지를 만들어줘.

규칙:
- 전체 레이아웃은 중앙 정렬형 컨테이너 기반
- 데스크톱 본문 폭은 1200px
- 모바일에서는 1열 스택 구조
- 한국어 산세리프 폰트 사용, 기본은 Pretendard 우선
- 메인 컬러는 블루, 보조로 연한 블루/그린/노랑/회색 사용
- 배경은 흰색 또는 아주 옅은 회색
- 사진은 실사 중심, 사람/현장/일상 톤
- 구분선은 1px의 옅은 회색
- 카드와 버튼은 단정하고 절제된 스타일
- 강한 대시보드 느낌보다 부드러운 정보형 기관 사이트 톤 유지
